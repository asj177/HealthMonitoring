# Gradle Information
1)Build on Gradle 2.3
2)Java version 1.8 
3)Install Gradle 
4)Go the Root Folder where build.gradle is present and run the command 
gradle build 
or 
gradle clean build 

5)Go to /build/libs,the Jar is created and run the command java -jar NameOfTheJAR

#Prerequisites 
1)lm-sensor
2)dmidecode(should be present)
